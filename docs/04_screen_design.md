## 4. レスポンシブ対応方針

| 画面 | PC | タブレット | スマホ |
|------|-----|----------|--------|
| 管理画面（主催者/装飾業者） | フル機能 | サイドメニュー折りたたみ | 基本閲覧のみ |
| 出展社ポータル | フル機能 | フル機能 | アップロード・状況確認に最適化 |
| AIレビュー画面 | 左右分割表示 | 上下分割表示 | タブ切り替え表示 |
| 協力会社ポータル | フル機能 | フル機能 | 一覧・詳細閲覧に最適化 |

| 項目 | 対応方針 |
|------|---------|
| カメラ起動 | スマホ・タブレットではネイティブカメラアプリを呼び出し（`<input type="file" capture="environment">`） |
| PC | Webカメラまたはファイル選択にフォールバック |
| プレビュー | 撮影後に画像プレビューを表示、再撮影・回転・トリミング操作が可能 |
| 複数撮影 | 連続撮影モード対応（書類の表裏・複数ページを続けて撮影可能） |
| ボタン配置 | スマホでは「カメラで撮影」ボタンを「ファイルを選択」と同サイズで並列配置し、タップしやすい大きさを確保 |
| 画質 | 撮影画像は自動的にOCR処理に適した解像度（300dpi相当）にリサイズ |
